# Brasil na Brasa

Este é um projeto de um site para o restaurante fictício "Brasil na Brasa", que oferece pratos típicos da culinária brasileira com foco em delivery. O site foi desenvolvido utilizando HTML, CSS e JavaScript.

## Estrutura do Projeto

- **HTML**: Estrutura do conteúdo.
- **CSS**: Estilização visual, incluindo uso de TailwindCSS para otimizações adicionais.
- **JavaScript**: Funcionalidades interativas, como carrinho de compras.
- **Imagens e Vídeos**: Localizados na pasta `public/assets`.

## Funcionalidades Principais

1. **Seção de Destaque (Hero):** Vídeo em loop apresentando o restaurante.
2. **Cardápio Interativo:**
   - Pratos principais com descrição, imagem e botão para adicionar ao carrinho.
   - Sobremesas destacadas em uma seção separada.
3. **Carrinho de Compras:**
   - Popup interativo mostrando itens selecionados e o valor total.
4. **Informações de Entrega:** Vídeo e texto destacando o foco em delivery.
5. **Footer com Redes Sociais:** Links para páginas sociais fictícias e informações de contato.

## Como Executar o Projeto

1. Clone este repositório:
   ```bash
   git clone https://github.com/matcbv/brasil_na_brasa.git
   ```

2. Navegue até a pasta do projeto:
   ```bash
   cd brasil_na_brasa
   ```

3. Abra o arquivo `index.html` em seu navegador.
